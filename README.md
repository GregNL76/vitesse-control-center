# Vitesse Control Center

Vitesse Control Center (VCC) is a professional Nintendo Switch library management system designed for Synology NAS.

## Planned features

- Library scanner
- Tinfoil synchronization
- Update detection
- SQLite database
- Health checks
- Dashboard
- Search & filters
- Cloudflare support

## Status

🚧 Under development

## Platform

- Synology DSM
- Python 3.8+
