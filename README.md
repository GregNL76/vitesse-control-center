# Vitesse Control Center (VCC)

VCC is a Nintendo Switch Library Management System designed for Synology NAS.

## Features (planned)

- Library Scanner
- SQLite Database
- Tinfoil Synchronization
- Health Check
- Dashboard
- Cloudflare Integration

## Requirements

- Python 3.8+
- Synology DSM (target platform)
